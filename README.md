# GunoApp
Gume — моб. приложение эмоциональный-дневник для отслеживания настроения, событий и психологического состояния.

# Стек
```
– Kotlin
– One Activity
– Retrofit2
– REST API 
– Glide
– Corutines
– MVVM, Hilt, Dagger2
– Room, SharedPreferences
– Livedata, ViewBinding
– Figma, Postman
```
